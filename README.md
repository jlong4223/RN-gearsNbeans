# Gears N' Beans React Native App

This is the mobile app for Gears N' Beans web.

## Installation

Versions:

- node: v16.0.0
- ruby: 3.0.0
- xcode: 13.1
- gradle: 6.7

iOS Sim:

- npm i
- npm run pods
- npm run ios

Android Sim:

- npm i
- npm run android

## Main Technologies:

- React Native
- NativeBase
- JavaScript
- Redux
- XCode
- Android Studio
- Git
